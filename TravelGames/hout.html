📋 How to Use

Wood Price Comparator — Travel Edition

  1. Select or Create a Seller — e.g. local garage, farm stall, depot. Optionally supply a website URL.
  2. Select or Create Wood Type — e.g. Sekelbos, Kameeldoring.
  3. Fill in quantity, bag weight, and total cost.
  4. Save the listing — the tool calculates Rand-per-kilogram for every entry.
  5. Tap any row in the compare table for full details and a link to the seller's website.

Deal Ratings:

  • Best Deal: Lowest R/kg in the registry.
  • Excellent Value: Within 10% of the best rate.
  • Good Deal: Within 25% of the best rate.
  • Suboptimal: More than 25% above cheapest.
