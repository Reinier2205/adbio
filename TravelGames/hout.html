📋 Game Rules & Help

How to use the Comparator:

  1. Select or Create a Seller (e.g., local garage, farm stall, depot). When adding, optionally supply their website URL.
  2. Select or Create Wood Type (e.g., Sekelbos, Kameeldoring).
  3. Fill in the package quantity, single bag weight, and total package cost.
  4. Save the listing. The calculator dynamically analyzes metrics to supply the Rand-per-kilogram yield.
  5. Tapping on any row inside the compact compare table displays comprehensive details and actions, including an instant portal link to the seller's web page if set!

Classification Values:

  • Best Deal: Lowest R/kg in the registry.
  • Excellent Value: Within 10% of the best rate.
  • Good Deal: Within 25% of the best rate.
  • Suboptimal: More than 25% expensive.
